# Pet Printables — Store Policies

Source-of-truth drafts for the six Shopify policies. Copy each file's content into the matching editor at:

**Shopify Admin → Settings → Policies**

| File | Shopify slot | Shopify renders it at |
|---|---|---|
| `contact-information.md` | Contact information (**required**) | `/policies/contact-information` |
| `privacy-policy.md` | Privacy policy | `/policies/privacy-policy` |
| `terms-of-service.md` | Terms of service | `/policies/terms-of-service` |
| `refund-policy.md` | Return and refund policy | `/policies/refund-policy` |
| `shipping-policy.md` | Shipping policy | `/policies/shipping-policy` |
| `legal-notice.md` | Legal notice | `/policies/legal-notice` |

## Placeholders to fill before pasting

Search each file for these tokens and replace:

- `[BUSINESS_LEGAL_NAME]` — the legal entity (e.g. "Pet Printables Inc." or the proprietor's legal name if sole proprietorship).
- `[BUSINESS_ADDRESS]` — registered mailing address. Use a commercial mailbox / registered-agent address if you don't want your home address public; BC requires an accessible address for consumers.
- `[BUSINESS_PHONE]` — optional but recommended; an answering service or VoIP line is fine.
- `[SUPPORT_EMAIL]` — e.g. `hello@petprintables.ca`.
- `[GST_NUMBER]` — your CRA GST/HST number once registered (required once you hit $30K in sales).
- `[DATE_EFFECTIVE]` — the effective date you paste these in.
